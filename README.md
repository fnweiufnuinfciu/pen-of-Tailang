# 太郎的笔

AI 长篇小说写作工具。从故事规划、角色设定到逐章生成，一站式完成创作。

## 功能

- **故事规划** — 输入需求，AI 生成章节大纲 + 角色卡
- **逐章生成** — 解析 → 确认 → 生成，实时进度 + 断点续传
- **动态状态锚点** — 自动追踪角色状态/位置/物品，防止情节矛盾
- **单章重生成** — 不满意某章可单独重写
- **角色卡编辑** — JSON 编辑器 + 可视化表单编辑
- **增强写作模式** — 可选的细节增强提示词
- **一键导出 TXT** — 全部章节拼接下载
- **纯浏览器存储** — 配置和大纲自动保存到 localStorage
- **零外部依赖** — 只有 Python 标准库

## 快速开始

```bash
python server.py
```

浏览器打开 `http://127.0.0.1:8765`

或双击 `start.bat`（Windows）。

## 工作流

1. **故事规划** → 输入剧情框架，AI 生成大纲和角色卡
2. **发送到生成器** → 自动跳转并解析章节
3. **确认章节目录** → 可在此编辑修改
4. **开始生成** → 逐章生成，右侧实时显示进度
5. **导出 TXT** → 下载全文

## 大纲格式

```
第1章 标题
内容要点...

第2章 标题
内容要点...
```
